# API de Gerenciamento de Autor

Serviço RESTful criado com **Spring Boot** para gerenciar os dados de Autores. O projeto exercita a construção de uma estrutura em camadas (Model, Repository, Service, Controller) e o mapeamento JPA com múltiplos campos de texto.

## 🚀 Tecnologias Utilizadas
* Java
* Spring Boot
* Spring Data JPA
* Banco de Dados Relacional

## 📌 Endpoints da API

A API expõe os seguintes endpoints na rota base `/Autores`:

* `POST /Autores`: Cria um novo Autor.
* `GET /Autores`: Lista todos os Produtos cadastrados.
* `GET /Autores/{id}`: Busca um Autor específico pelo seu ID.
* `DELETE /Autores/{id}`: Deleta um Autor específico pelo seu ID.
